#
    航空货运货物的体积重量（公斤）=货物的体积（长（cm）×宽（cm）×高（cm））/6000
    DHL，UPS更改体积重计算标准。
    DHL与UPS快件由2009年2月1日起，体积重的计算标准由6000改为5000，具体如下：
    如体积尺寸：100cm x 100cm x 100cm的快件。
    旧体积重计算： 100x100x100/6000=166.67KG 计费重量167KG。
    新体积重计算： 100x100x100/5000=200KG 计费重量200KG。
